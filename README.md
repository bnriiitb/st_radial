# st-radial
A streamlit component to display metrics neatly </br>
[![GitHub][github_badge]][github_link] [![PyPI][pypi_badge]][pypi_link] 

## Installation

```sh
pip install st-radial
```

## Getting started

```python
from st-radial import st-radial

st-radial('Some metric', value=80)
```
